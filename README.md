# Le Pokedex à Coco !

Voici mon projet de Pokédex réalisé dans le cadre de mon cours de développement web. 
L'objectif principal était de manipuler une API en JavaScript (PokeAPI), et j'ai décidé de faire une interface sous forme de GameBoy Color avec un peu de CSS.

## Fonctionnalités

* Barre de recherche pour trouver un Pokémon (par nom ou ID).
* Affichage de l'image, du nom et du type sur l'écran de la console.
* Le bouton "A" permet de voir les statistiques du Pokémon, et le bouton "B" sert à faire retour.
* La croix directionnelle affiche un Pokémon au hasard.
* Un panneau latéral avec un historique (5 dernières recherches) et 3 suggestions de Pokémon aléatoires.

## Technologies utilisées

* HTML / CSS
* JavaScript (utilisation de `fetch()` pour appeler https://pokeapi.co/)

## Comment tester le projet ?

Le projet fonctionne sans serveur. Il suffit de télécharger les fichiers et d'ouvrir le fichier `index.html` dans n'importe quel navigateur web.
Ou alors d'aller sur le lien web suivant : https://cocodelprado.github.io/Pokedex/
